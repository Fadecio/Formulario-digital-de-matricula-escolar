# Formulário Digital de Matrícula Escolar

## Visão Geral
Sistema web de matrícula escolar desenvolvido para digitalizar e organizar o processo de cadastro de alunos.

A aplicação possui validações inteligentes, campos condicionais e foco em experiência do usuário, tornando o preenchimento mais seguro, rápido e intuitivo.

---

## Objetivo
Criar uma solução moderna para matrícula escolar digital, permitindo:

- Cadastro completo de alunos
- Validação automática dos dados
- Melhor experiência no preenchimento
- Redução de erros em formulários
- Interface acessível e responsiva

---

## Funcionalidades

### Cadastro do Aluno
- Nome completo
- Data de nascimento
- CPF do responsável
- Série/Turma
- Informações de contato

### Validações Inteligentes
- Verificação de campos obrigatórios
- Máscaras de preenchimento
- Validação de idade mínima
- Bloqueio de datas futuras
- Feedback visual em tempo real

### Campos Condicionais
- Exibição dinâmica de campos
- Alteração do formulário conforme seleção do usuário

### Feedback ao Usuário
- Toasts informativos
- Mensagens de erro amigáveis
- Destaque visual para campos inválidos

### Interface
- Layout responsivo
- Navegação intuitiva
- Experiência otimizada para desktop e mobile

---

## Requisitos Funcionais

### RF001
O sistema deve permitir cadastrar informações do aluno.

### RF002
O sistema deve validar os dados antes do envio.

### RF003
O sistema deve impedir datas de nascimento futuras.

### RF004
O sistema deve validar a idade mínima permitida.

### RF005
O sistema deve exibir mensagens de feedback ao usuário.

### RF006
O sistema deve possuir campos condicionais dinâmicos.

---

## Requisitos Não Funcionais

### RNF001
A aplicação deve ser responsiva.

### RNF002
O sistema deve possuir boa usabilidade.

### RNF003
O código deve ser modular e organizado.

### RNF004
A aplicação deve funcionar diretamente no navegador.

### RNF005
As validações devem ocorrer em tempo real sempre que possível.
